# Lista de Tarefas

Aplicação web para gerenciamento de tarefas e responsáveis, desenvolvida com Java, JSP e MySQL.

## Tecnologias

- Java (Servlets / JSP)
- MySQL
- HTML/CSS

## Banco de dados

Execute o arquivo `tarefas.sql` para criar o banco e as tabelas necessárias.

## Como executar

1. Configure a conexão com o banco em `src/java/util/Conexao.java`
2. Execute `tarefas.sql` no MySQL
3. Faça o deploy da aplicação em um servidor compatível (ex: Apache Tomcat)
4. Acesse `index.html` no navegador
